# Hostel Management System

A simple Hostel Management System for managing hostel activities like student records, complaints, mess, maintenance, and administration.

---

## Roles in System

### Student
- Register and login
- View room details
- Apply for hostel services
- Raise complaints

###  Warden
- Manage student records
- Approve room allocations
- Monitor hostel activities

### Mess Incharge
- Manage food menu
- Handle mess related complaints
- Update daily meals

### Technician
- Handle maintenance requests
- Fix hostel infrastructure issues
- Update repair status



## How to Run the Project

### fill all .env file content

### 1. Clone the Repository
```bash
git clone https://github.com/ravikumarstm333/Hostel-Management-System.git
cd Hostel-Management-System
```
### Run Backend
```bash
cd hostelc-backend
pip install -r requirements.txt
python run.py
```


### run frontend
```bash
cd hostelc-student
npm install
npm start
```
